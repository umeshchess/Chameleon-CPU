# Chameleon CPU 🦎⚡

An 8-bit CPU designed from scratch by Umesh Penkey (JNTUACEK)

## Current Status
- [x] ALU — Complete ✅
- [ ] Registers
- [ ] Control Unit
- [ ] Program Counter
- [ ] Complete CPU

## ALU Features
- 17 operations via 5-bit OpCode
- Arithmetic: ADD, ADC, SUB, SBB, ONC, TWC
- Logic: AND, OR, XOR, XNOR
- Shifts: LSL, LSR, ASR
- Rotates: ROL, ROR, RCL, RCR
- Flags: Zero, Negative, Overflow, Carry
